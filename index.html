Distributed Login (In Development)

  • My current final year project which is still in development. It's a two-factor authentication system that distributes the key accross SQL databases and a USB device.

  • Therefore if one database is compromised, the entire authentication key is not compromised, at least in theory.

  • It is also designed to combat Man-in-the-Browser (MitB) attacks by changing the key at sign out.
