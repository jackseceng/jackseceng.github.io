Security Toolbox

  1. Home
  2. Projects
  3. Security Toolbox

Project Aims

  • Create tools for various different offensive/defensive security requirements
  • Utilised github as a method of quickly acquiring these tools wherever there is a web connection
  • Spin off any ideas in other projects

Features

  • RSA Calculator: A script that helps calculate the various elements of RSA
  • Single Site App: A python based web client that display a pre-determined web page, useful for SOC monitoring
  • Status Script: A simple bash script that can tell you useful information about your Debian based Linux box.

  • RSA Calculator

    Allows you to enter each part of RSA calculation: n, m, c and tot

    D part of calculation is still a work in progress

    Written in Python 2.7

  • Single Site App

    Dedicated web interface display app, no search bar or browser controls

    Ideal for keeping tabs on a SoC dashboard for example

    Does not support CSS or HTML5 or any dynamic web plugins

    NOTE: This project borrowed code from Paweł Miech (see links below)

  • Status Script

    Bash script for displaying hardware and configuration info for Debian based linux distros

    Currently only tested on Ubuntu distros: 14.04, 16.04 & 18.04
