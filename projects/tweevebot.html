TweeVEBot

  1. Home
  2. Projects
  3. TweeVEBot

Project Aims

  • Become a go to source of vulnerability related news on Twitter
  • Tweet out useful information from CVE sites in an easily readable form
  • Have multiple different tweet formats for different types of vulnerability

Features

  • Gathers data from JSON feeds from >cvedetails.com
  • Tweets top 3 CVEs, with and 8+ CVSS score. Includes exploit count & url
  • Second version to make detailed attack tweets based on 4 attack types
  • Types are: SQLi, Privilege Escalation, DDoS & Remote Code Execution

  • Top 3 example

    Lists in order of most recently updated CVE

    Generates tinyurl for each CVE page due to tweet character limit

    Uses #Cybersecurity so it can be found by twitter search

  • Detailed exploit example (WIP)

    Details the most recently updated, highest scoring CVE

    Related to one of the 4 chosen categories, in this example the type is remote code execution

    Some use of puns and emojis to spice the tweet up a bit ;)
